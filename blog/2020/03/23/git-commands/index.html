常用git命令记录

  • command
  • git

记录工作中经常用到的git命令，记不清的时候，可以使用命令git --help

2020-03-23

1、新建仓库

#当前目录下创建一个git仓库
git init
#当前目录下创建一个名为repository的git仓库
git init [repository]
#克隆一个远程git仓库，包括全部提交记录
git clone [repository url]

2、操作仓库分支

#查看本地分支
git branch
#查看远程分支
git branch -r
#查看本地+远程全部分支
git branch -a
#新建分支
git branch [branch-name]
#删除分支
git branch -d [branch-name]
#切换分支
git checkout [branch-name]
#新建分支并切换
git checkout -b [branch-name]
#合并分支至当前分支
git merge [branch-name]
#删除远程分支，慎重！！！
git push origin --delete [branch-name]

3、操作仓库文件

#增加当前目录下的文件
git add [file1] [file2]
#增加当前目录下的目录
git add [dir]
#增加当前目录下的所有文件
git add .
#删除文件
git rm [file1] [file2]
#文件改名
git mv [file-origin] [file-new]
#重置文件更新
git reset [file]

4、仓库更新提交操作

#提交更新
git commit -m [message]
#提交指定文件更新
git commit [file1] [file2] -m [message]
#查看commit可选命令项
git commit -help
#变化暂存
git stash
#变化暂存载入
git stash pop

5、本地仓库与远程仓库同步操作

#下载远程仓库的所有变动
git fetch [remote]
#显示所有远程仓库
git remote -v
#显示某个远程仓库的信息
git remote show [remote]
#增加一个新的远程仓库，并命名
git remote add [shortname] [url]
#取回远程仓库的变化，并与本地分支合并
git pull [remote] [branch]
#上传本地指定分支到远程仓库
git push [remote] [branch]
#强行推送当前分支到远程仓库，即使有冲突
git push [remote] --force
#推送所有分支到远程仓库
git push [remote] --all

6、变更信息查看

#显示变更
git status
#显示提交历史
git log
#工作区和暂存区的差异
git diff
#显示某次更新
git show [commit]
